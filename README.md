Короткое описание проекта и вложенных в него файлов 

- APP - главная директория нашего апи 
-- api -> v1 -> Ручки для приема/отправки запросов 
-- consumers -> работа с очередью
-- core -> конфиги
-- db -> инициализация доступа к базе данных и скрипт создания/очистки бд
-- models -> модели базы данных
-- schemas -> pydantic схемы по валидации данных
-- services -> бизнес логика, взаимодействие с базой данных, патерн гарантии доставки ответа 

Слой работы с базой данных находится именно в папке services
